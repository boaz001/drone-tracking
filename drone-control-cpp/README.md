# drone-control-cpp

## Description:
Outer control loop of the drones

## Requires:
    git
    CMake >= 2.8
    OpenCV >= 2.4
    boost >= 1.39
    Autodesk FBX SDK >= 2015.1
    OptiTrack NatNet SDK >= 2.7

## Install
Build the drone-control demo

    cd ~/code/drone-tracking/drone-control-cpp
    mkdir build
    cd build
    cmake ..
    make
    make install
